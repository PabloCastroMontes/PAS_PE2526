# PAS_PE2526
Simulación y visualización de un proceso estocástico: el paseo aleatorio simple.

## DESCRIPCIÓN DEL PROYECTO
El objetivo principal de este proyecto es poder simular y visualizar diferentes trayectorias del paseo aleatorio simple (PAS), un proceso estocástico que se define de la siguiente manera:


  Dado $p \in [0,1]$ , para cada $t \in \mathbb{N}$ se considera

  $$X_t = \sum_{s=1}^{t} Z_s$$


  siendo $\{Z_s\}$, ${s \in \mathbb{N}}$, variables aleatorias independientes e idénticamente distribuidas según la siguiente distribución:


  $$P(Z_s = 1) = p, \quad P(Z_s = -1) = 1 - p.$$
